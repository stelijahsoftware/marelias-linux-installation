choose 'recovery mode' # if recovery screen does not exist, use advanced settings, see next file. put it in fstab. put danger.

Reboot
Choose "Advanced options for ubuntu"
Press "e"
Find the line that starts `linux /boot/vmlinuz-...`
Replace `ro quiet splash` with `rw init=/bin/bash`
Press F10
Now you can either `vim /etc/fstab` or `passwd username` to change sudo password
